# DecodeLabs Data Analytics Internship - Project 1

## Data Cleaning & Preparation

### Overview
This project was completed as part of the DecodeLabs Data Analytics Internship Program.

The objective was to inspect, clean, and prepare a raw dataset for further analysis by identifying missing values, checking duplicates, validating formats, and ensuring data quality.

---

## Objectives

- Identify missing values
- Handle incomplete data
- Check duplicate records
- Verify data consistency
- Prepare a clean dataset for analysis

---

## Dataset Summary

The dataset contained customer order information including:

- Order ID
- Customer Information
- Product Details
- Quantity
- Pricing Information
- Coupon Codes
- Order Dates

---

## Data Cleaning Process

### 1. Missing Value Detection

The CouponCode column contained missing values.

### 2. Missing Value Handling

All missing CouponCode values were replaced with:

NO_COUPON

This indicates that no promotional coupon was used during the transaction.

### 3. Duplicate Verification

The dataset was checked for duplicate records.

Result:
- No duplicate rows found.
- No duplicate Order IDs found.

### 4. Data Validation

The dataset was reviewed to ensure:

- Consistent date formatting
- Correct numerical values
- Proper text formatting
- Accurate data structure
